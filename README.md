# Active Directory Basics Lab
## Overview
Completed an Active Directory fundamentals lab focused on domain environments, authentication, organizational units, and machine accounts.
This lab was performed using TryHackMe and supplemented with additional research and documentation.
## Objectives
- Understand Active Directory structure
- Learn how domain authentication works
- Identify users, groups, and machine accounts
- Practice Active Directory enumeration concepts

## Topics Covered

### Domain Controllers
Learned how Domain Controllers:
- Authenticate users
- Enforce policies
- Store AD objects
- Manage domain resources

### Organizational Units (OUs)
Explored how OUs are used to:
- Organize users and computers
- Apply Group Policies
- Delegate administration

### Machine Accounts
Learned that computer accounts in Active Directory end with `$`.

Example:
```text
TOM-PC$
```

### Authentication
Studied:
- Kerberos authentication
- NTLM authentication
- Ticket Granting Tickets (TGTs)
